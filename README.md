# 赤ちゃんイベント計算アプリ

赤ちゃんの生年月日と性別を入力すると、
お七夜・お宮参り・お食い初めなどのイベント日を自動で計算して表示するWebアプリです。

## アプリ概要

育児中の家庭向けに、赤ちゃんの成長イベントの日付を簡単に確認できるようにすることを目的としたアプリです。
生年月日と性別を入力するだけで、代表的な行事の日程とその簡単な説明を一覧表示します。

## 主な機能

* 生年月日入力
* 性別選択
* 赤ちゃんイベントの日付自動計算
* イベント説明表示
* 結果ページ表示

表示されるイベント例

* お七夜
* お宮参り
* 100日祝い（お食い初め）
* ハーフバースデー
* 1歳の誕生日
* 初節句

## 使用技術

* HTML
* CSS
* JavaScript

※ フロントエンドのみで動作するシンプルなWebアプリです。


## 工夫した点

* JavaScriptで日付計算を実装し、育児イベントの日程を自動生成
* 初節句は誕生日より後に来る最初の節句になるようロジックを実装
* シンプルで見やすいUIを意識したデザイン
